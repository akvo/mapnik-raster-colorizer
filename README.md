# mapnik-raster-colorizer

Example of Mapnik raster colorizer

## Usage

    docker-compose up --build -d

### Run test

    docker-compose run mapnik /usr/local/src/run.sh test.js
    docker-compose run mapnik /usr/local/src/run.sh test2.js
